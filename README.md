#### 安装流程

- 在页面右侧*Releases*中下载对应操作系统的最新版本安装程序
- 用户双击安装包进行安装(杀毒软件可能拦截)
- 
#### 支持功能
- [x] 开机自动连接
- [x] 断线重连
- [x] 自定义网关IP及端口
- [x] 多平台支持

#### 存在问题
- 长期显示主界面会导致内存占用率暴增
- 解决方案：工具切换至后台，需要时在任务栏图标中打开，内存占用稳定在55M
